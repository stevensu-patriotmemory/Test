# 失落的方舟 副本攻略

這是一個為《失落的方舟》玩家設計的副本攻略網站，提供詳細的副本收益數據和攻略信息。

## ✨ 功能特色

- 📊 **完整的副本數據表** - 顯示各副本的收益信息
- 🔍 **實時搜索功能** - 快速查找副本
- 🎯 **難度篩選** - 按難度（獵人/普通/困難）篩選副本
- 📱 **響應式設計** - 支持桌面和移動設備

## 🚀 快速開始

1. 克隆或下載本倉庫
2. 用瀏覽器打開 `index.html` 文件
3. 開始探索副本攻略！

## 📖 使用方法

- **搜索副本**：在搜索框輸入副本名稱
- **篩選難度**：使用下拉菜單選擇難度
- **查看數據**：表格顯示各開口和總收益
